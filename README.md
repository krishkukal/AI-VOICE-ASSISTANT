# AI Hybrid Voice Assistant

A powerful, intelligent **hybrid voice assistant** that listens to your voice commands, also supports text input, answers questions using **Groq AI**, and speaks back responses using **Windows Speech Engine or pyttsx3 fallback**. Built with Python, speech recognition, and Groq's lightning-fast LLM.

## Features

* 🎙️ **Voice & Text Input** - Speak naturally or type your commands
* 🔊 **Voice Output** - Uses Windows Speech Engine with pyttsx3 fallback
* 🤖 **Groq AI Integration** - Get intelligent answers to any question (FREE)
* 🚀 **Lightning Fast** - Powered by Groq's ultra-fast inference
* 📅 **Basic Commands** - Time, date, jokes, greetings
* 📂 **Open Applications & Folders** - Launch Notepad, Calculator, Paint, CMD, PowerShell, Chrome, Desktop, Downloads, Documents
* 🌐 **Open Websites** - Open websites directly from commands
* 🔄 **Switch Modes** - Toggle between voice and text input anytime
* 💻 **Shutdown & Restart PC** - Safe confirmation with 10-second cancel countdown

## 🎯 Commands

| Command           | Description                                           |
| ----------------- | ----------------------------------------------------- |
| `time`            | Get current time                                      |
| `date`            | Get current date                                      |
| `joke`            | Tell a random joke                                    |
| `hello`           | Greeting                                              |
| `who made you`    | About the creator (Krish Kukal from Tecnia Institute) |
| `open notepad`    | Open Notepad                                          |
| `open calculator` | Open Calculator                                       |
| `open paint`      | Open Paint                                            |
| `open cmd`        | Open Command Prompt                                   |
| `open powershell` | Open PowerShell                                       |
| `open chrome`     | Open Chrome browser                                   |
| `open desktop`    | Open Desktop folder                                   |
| `open downloads`  | Open Downloads folder                                 |
| `open documents`  | Open Documents folder                                 |
| `play [song]`     | Play song on YouTube                                  |
| `search [query]`  | Search Google                                         |
| `shutdown`        | Shutdown computer safely                              |
| `restart`         | Restart computer safely                               |
| `cancel`          | Cancel shutdown/restart                               |
| `text mode`       | Switch to typing mode                                 |
| `voice mode`      | Switch to voice mode                                  |
| `help`            | Show commands                                         |
