#AI Voice Assistant

A powerful, intelligent voice assistant that listens to your voice commands, answers questions using Groq AI, and speaks back responses. Built with Python, speech recognition, and Groq's lightning-fast LLM.

## Creator

**This assistant was created by Krish Kukal from Tecnia Institute**

- Designed to help people with daily tasks using natural voice commands
- Powered by Groq's ultra-fast AI inference

## Features

- 🎙️ **Voice & Text Input** - Speak naturally or type your commands
- 🔊 **Voice Output** - Hear responses spoken aloud
- 🤖 **Groq AI Integration** - Get intelligent answers to any question (FREE)
- 🚀 **Lightning Fast** - Powered by Groq's ultra-fast inference
- 📅 **Basic Commands** - Time, date, jokes, greetings
- 📂 **Open Applications** - Launch apps like Notepad, Calculator, Chrome
- 🌐 **Open Websites** - Open Google, YouTube, GitHub and more
- 🔄 **Switch Modes** - Toggle between voice and text input anytime

## 🎯 Commands

| Command | Description |
|---------|-------------|
| `time` | Get current time |
| `date` | Get current date |
| `joke` | Tell a random joke |
| `hello` | Greeting |
| `who made you` | About the creator (Krish Kukal from Tecnia Institute) |
| `open notepad` | Open Notepad |
| `open calculator` | Open Calculator |
| `open paint` | Open Paint |
| `open chrome` | Open Chrome browser |
| `open google` | Open Google website |
| `open youtube` | Open YouTube |
| `play [song]` | Play song on YouTube |
| `search [query]` | Search Google |
| `text mode` | Switch to typing mode |
| `voice mode` | Switch to voice mode |
| `help` | Show commands |
| `exit` | Quit assistant |

**Ask anything!** - "What is machine learning?", "Who is India's President?", "Tell me a fun fact", "Explain quantum physics"

### Prerequisites
